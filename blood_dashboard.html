Blood Donation Campaign Dashboard

March 2025 - April 2025
Python Geospatial Analysis Data Visualization Machine Learning API Development

Project Overview

This project involved designing and developing An interactive dashboard for analyzing and optimizing blood donation campaigns using Python.

Technical Approach

This dashboard visualizes donor distribution, health conditions, donor retention, campaign effectiveness, and predicts eligibility using machine learning models.

  • Map Donor Distribution: Visualizes donor locations using interactive maps.
  • Health Conditions & Eligibility: Charts showing eligibility based on health conditions.
  • Profiling Ideal Donors: Uses clustering to find ideal donor characteristics.
  • Campaign Effectiveness: Tracks donation trends and behaviors over time.
  • Donor Retention: Analyzes repeat donor patterns.
  • Sentiment Analysis: Analyzes feedback from donors.
  • Eligibility Prediction API: Predicts if a donor is eligible based on health and demographic data.

How to Run

  1. Clone the repository: git clone https://github.com/PagnaMMA/Blood-Campaign-Dashboard.git
  2. Navigate to the project folder: cd blood-donation-dashboard
  3. Install dependencies using pip install -r requirements.txt.
  4. Run the api with uvicorn api:app --reload.
  5. Run the dashboard with streamlit run dashboard.py.
  6. Access the API at http://localhost:5000/predict.
